### env 파일 구성


```
SERVER_PORT=
DATABASE_URL=
HASHROUNDS=
ACCESS_TOKEN_SECRET =
REFRESH_TOKEN_SECRET =
```

### 프로젝트 초기 설정
```
yarn                  // 패키지 설치
yarn prisma:seed      // 초기 데이터 구성 
```

### 프로젝트 시작
```
yarn start
```

### 프로젝트 테스트
```
yarn test
```

### 프로젝트 스웨거
```
퍼블릭 IP:3000 / api-docs
```
